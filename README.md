# 📅 Schedulo - Daily Task Scheduler

> A Java-based Daily Task Scheduler application designed to help users organize, manage, and track daily activities efficiently. The system allows users to schedule tasks, update or remove them, and monitor task completion status through a simple and user-friendly interface.

---

## 📌 Project Description

The Daily Task Scheduler is developed using Java to improve personal productivity and task management. Users can create tasks with specific dates and times, edit schedules, mark completed tasks, and maintain daily activity records.

This project demonstrates the implementation of core Java concepts such as Object-Oriented Programming (OOP), data structures, file handling, and optional database connectivity.

---

## 🚀 Features

* ➕ Add new tasks with date and time
* 📝 Update existing tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 📅 View daily scheduled tasks
* ⏰ Task reminder and scheduling support
* 📂 Store task records using file handling
* 🔍 Search and filter tasks
* 🛠 Simple and user-friendly interface
